# N46Whisper For Lecture

[![Google Colab で開く](https://colab.research.google.com/assets/colab-badge.svg)]
(https://colab.research.google.com/drive/1keQeknae1mxEA4XTGtZUj8hSe1vGNHog?usp=sharing)

N46Whisper For Lecture は、講義録音やオンライン教材動画の文字起こし・
字幕生成を効率化するための Google Colab ノートブックです。

本プロジェクトは、乃木坂46グループの字幕制作効率化を目的とした
オープンソースプロジェクト **N46Whisper** (https://github.com/Ayanaminn/N46Whisper) をベースにしています。
本バージョンでは、ファンサブ用途ではなく、
**大学講義・学習用途向け** に機能を改良しました。

学生であり、有料ツールや会員制サービスを継続的に利用することが難しかったため、
無料で利用でき、環境構築の手間も少ない Google Colab を実行環境として採用しました。

## 開発の背景
大学の講義やオンライン教材は復習に時間がかかるため、
検索可能な文字起こしを作成することで学習効率を向上させることを目的として
本プロジェクトを作成しました。

生成した文字起こしは、Markdown ベースのノートツール（Obsidian）や
言語モデルと組み合わせて、講義内容の理解や演習問題の解答作成に活用しています。

## 主な改良点
オリジナルの N46Whisper に対して、以下の改良を行いました。

- **Colab 上での録音機能追加**
  - ブラウザ上で講義音声を録音し、そのまま同一ノートブック内で
    文字起こしが可能。

- **複数出力形式への対応**
  - 従来の ASS / SRT に加え、**Markdown 形式での文字起こし**を追加。
  - 学習用途向けに、タイムスタンプを簡潔な形式に変更。
  - Obsidian 等のナレッジ管理ツールで扱いやすい構成。

- **出力ファイルの整理**
  - 生成されるすべてのファイルを **1つの ZIP ファイル** にまとめ、
    ディレクトリ構造を整理。

- **コード整理**
  - 個人利用・学習用途に不要な履歴や冗長な構造を削除し、
    可読性と保守性を向上。

## 利用例
- 大学講義（例：量子力学）の録音・文字起こし
- オンライン講座動画の文字起こし
- 講義内容の復習
